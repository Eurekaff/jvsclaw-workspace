# 部署说明 (Runbook)

## 环境要求
- Docker 20.10+
- Docker Compose 2.0+
- 2GB+ 内存
- 10GB+ 磁盘空间

## 快速启动

### 1. 克隆仓库
```bash
git clone <repo-url>
cd enterprise-registry-search
```

### 2. 配置环境变量
```bash
cp .env.example .env
# 编辑 .env 填写 API key 等配置
```

### 3. 启动服务
```bash
docker-compose up -d
```

### 4. 验证部署
```bash
curl http://localhost:8000/health
# 应返回：{"status": "ok"}
```

### 5. 访问应用
- 前端：http://localhost:3000
- 后端 API: http://localhost:8000
- API 文档：http://localhost:8000/docs

## 日常运维

### 查看日志
```bash
docker-compose logs -f backend
docker-compose logs -f frontend
```

### 重启服务
```bash
docker-compose restart
```

### 停止服务
```bash
docker-compose down
```

### 数据备份
```bash
docker-compose exec backend tar -czf /backup/data.tar.gz /app/data
```

## 故障排查

### 后端无法启动
1. 检查日志：`docker-compose logs backend`
2. 验证端口占用：`lsof -i :8000`
3. 检查数据库连接

### 前端无法访问
1. 检查容器状态：`docker-compose ps`
2. 查看前端日志
3. 验证后端 API 可达性
