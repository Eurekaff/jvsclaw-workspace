## 模块划分

- **Excel Parser**: 解析用户上传的 Excel 文件，处理各种格式变体
  - 接口：`parse_excel(file) -> List[RawRecord]`
- **Entity Recognizer**: 从原始记录中识别企业主体
  - 接口：`recognize(raw_records) -> List[Entity]`
- **Registry API Client**: 调用查册接口获取企业信息
  - 接口：`search(entities) -> List[SearchResult]`
- **Review Manager**: 管理人工复核流程
  - 接口：`submit_review(entity_id, decision, notes)`
- **Export Service**: 导出结果为 CSV/Excel
  - 接口：`export(results, format) -> file`

## API 接口

- `POST /api/upload`: 上传 Excel 文件
- `GET /api/entities`: 获取识别后的实体列表
- `POST /api/search`: 批量查册
- `GET /api/results`: 获取查册结果
- `POST /api/review`: 提交复核决定
- `GET /api/export`: 导出结果
