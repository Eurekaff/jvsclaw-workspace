## 🤖 Agent 矩阵

### 1. ContentCreator Agent (主 Agent)

**职责**: 统筹内容创作全流程

**输入**:
```json
{
  "topic": "AI 工具推荐",
  "target_audience": "打工人",
  "selling_points": [
    "提高效率 10 倍",
    "节省每天 2 小时",
    "上手简单"
  ],
  "platforms": ["xiaohongshu", "weibo", "twitter"]
}
```

**输出**:
```json
{
  "topic": "AI 工具推荐",
  "platforms": ["xiaohongshu", "weibo", "twitter"],
  "versions": {
    "xiaohongshu": {...},
    "weibo": {...},
    "twitter": {...}
  },
  "summary": {
    "total_platforms": 3,
    "avg_score": 8.8
  }
}
```

**工作流程**:
1. 生成核心内容 (真人化)
2. 调用 PlatformAdapter 适配
3. 调用 AuthenticityChecker 检查
4. 优化未通过的内容
5. 输出最终版本

---

### 2. PlatformAdapter Agent (平台适配)

**支持平台**: 7 个

**核心能力**:
- ✅ 标题生成
- ✅ 语气调整
- ✅ emoji 智能添加
- ✅ 标签生成
- ✅ 脚本生成

**适配示例**:
```
核心内容 → 小红书版本
- 添加 emoji (10-30 个)
- 改为姐妹语气
- 添加标签 (5-10 个)
- 控制字数 (300-800)
```

---

### 3. AuthenticityChecker Agent (真人化检查)

**4 维度检查**:
1. 语言 (AI 词汇/真人词汇/情绪词)
2. 内容 (故事/细节/对比/缺点)
3. 互动 (称呼/提问/引导)
4. 格式 (emoji/标签/分段)

**评分标准**:
- ≥7 分：可发布
- <7 分：需优化

**AI 味检测**:
- AI 词汇库：20+ 词
- 真人词汇库：20+ 词
- 情绪词库：15+ 词

## 🚀 使用示例

### 基础使用

```python
from agents.content_creator.agent import ContentCreatorAgent

agent = ContentCreatorAgent("/path/to/workspace")

result = agent.execute({
    "topic": "AI 工具推荐",
    "target_audience": "打工人",
    "selling_points": [
        "提高效率 10 倍",
        "节省每天 2 小时"
    ],
    "platforms": ["xiaohongshu", "weibo", "twitter"]
})

# 获取各平台版本
xiaohongshu_content = result["platform_versions"]["xiaohongshu"]
weibo_content = result["platform_versions"]["weibo"]
twitter_content = result["platform_versions"]["twitter"]
```

### 工作流集成

```python
from workflows.content_creator_workflow import ContentCreatorWorkflow

workflow = ContentCreatorWorkflow()

# 定时生成内容
workflow.run_daily(
    topics=["AI 趋势", "工具推荐", "效率技巧"],
    platforms=["xiaohongshu", "weibo"],
    auto_publish=True
)
```

### CLI 使用

```bash
# 创建内容
claw content create \
  --topic "AI 工具推荐" \
  --audience "打工人" \
  --platforms xiaohongshu,weibo,twitter

# 查看结果
claw content list

# 发布内容
claw content publish --platform xiaohongshu --id 123
```
