## 📱 主流平台风格详解

### 1. 小红书 (Xiaohongshu)

**用户画像**:
- 年龄：18-35 岁为主
- 性别：女性占比 70%+
- 特点：追求品质、喜欢分享、信任真实体验

**内容特点**:
| 维度 | 要求 | 示例 |
|------|------|------|
| **字数** | 300-800 字 | 不宜过长 |
| **标题** | emoji+ 关键词 + 痛点 | "😭熬夜党必看！这个精华真的绝了" |
| **语气** | 姐妹聊天式 | "姐妹们！我真的要安利这个！" |
| **emoji** | 大量使用 | 每句 1-2 个 |
| **标签** | 5-10 个 | #护肤 #熬夜急救 #好物分享 |
| **配图** | 3-9 张精美图 | 首图最重要 |

**爆款公式**:
```
😱 痛点/惊讶开头 + 个人体验 + 产品/方法介绍 + 效果对比 + 强烈推荐
```

**示例文案**:
```
😭熬夜党姐妹看过来！这个精华真的绝了！！

姐妹们！作为一个常年熬夜的打工人
我真的要按头安利这个精华！！💯

之前天天熬夜到凌晨 2 点
脸色蜡黄 毛孔粗大 整个人老十岁😢
抱着试试看的心态买了这个
结果！！用了一周直接真香了！！

✨使用感受：
质地很清爽 不粘腻 吸收超快
早上起来脸不黄了 毛孔也细了
同事都问我是不是去做了医美😂

价格也很香！学生党冲！
我已经回购第三瓶了！！

#护肤 #熬夜急救 #精华推荐 #好物分享 #护肤心得
```

**避雷指南**:
- ❌ 不要像官方广告
- ❌ 不要堆砌专业术语
- ❌ 不要没有个人体验
- ✅ 要像闺蜜聊天
- ✅ 要有真实使用感受
- ✅ 要有前后对比

### 2. 抖音 (Douyin/TikTok)

**用户画像**:
- 年龄：16-40 岁
- 特点：注意力短暂、追求刺激、喜欢反转

**内容特点**:
| 维度 | 要求 | 示例 |
|------|------|------|
| **时长** | 15-60 秒 | 前 3 秒定生死 |
| **开头** | 强钩子 | "千万别买这个！除非你想..." |
| **节奏** | 快 | 2 秒一切 |
| **BGM** | 热门音乐 | 蹭热点 |
| **字幕** | 大字体 | 关键词高亮 |
| **互动** | 引导评论 | "你觉得呢？评论区见" |

**爆款公式**:
```
3 秒钩子 + 痛点放大 + 解决方案 + 效果展示 + 互动引导
```

**脚本示例**:
```
【0-3s】钩子
⚠️千万别买这个！除非你想变白！

【3-10s】痛点
是不是天天熬夜 脸色蜡黄？
用了很多产品都没用？

【10-30s】解决方案
试试这个！我用了 7 天
（展示使用过程）

【30-50s】效果
看！这是之前的我（黑黄）
这是现在的我（白皙）

【50-60s】互动
想知道是什么吗？
评论区告诉我！
```

**避雷指南**:
- ❌ 开头不要废话
- ❌ 节奏不要太慢
- ❌ 不要没有反转
- ✅ 前 3 秒必须抓住
- ✅ 要有视觉冲击
- ✅ 要引导互动

| **语气** | 专业 + 亲和 | 有深度但不高冷 |

**爆款公式**:
```

```

**示例结构**:
```markdown

## 开头（引发共鸣）
去年这个时候，我也是一个迷茫的打工人...

## 第一部分：故事/案例
那是一个普通的周一，我接到了 HR 的电话...

## 第二部分：深度分析
为什么会出现这种情况？我总结了 3 点...

## 第三部分：实用建议
如果你也遇到类似问题，建议这样做：
1. ...
2. ...
3. ...

## 结尾：金句总结
成长，就是不断试错的过程。
愿我们都能成为更好的自己。

---
👇 点击阅读原文，获取完整资料
```

**避雷指南**:
- ❌ 不要标题党（内容要匹配）
- ❌ 不要纯广告
- ❌ 不要没有价值
- ✅ 要有深度
- ✅ 要有个人故事
- ✅ 要有实用价值

## 🎯 真人化写作技巧

### 1. 个人故事法

**公式**: 时间 + 地点 + 人物 + 冲突 + 解决

**示例**:
```
❌ AI 味：这款产品非常优秀，值得购买。

✅ 真人味：上周三加班到凌晨 2 点，
          回家路上刷到这个，
          抱着试试看的心态买了，
          结果一周后同事都问我是不是去做了医美。
```

### 2. 情绪表达法

**公式**: 情绪词 + 具体感受 + 对比

**示例**:
```
❌ AI 味：效果显著，用户满意。

✅ 真人味：真的绝了！！😭
          我之前试过十几款产品都没用，
          这个直接用空瓶了！！
```

### 3. 细节描写法

**公式**: 五感描述 + 具体场景

**示例**:
```
❌ AI 味：质地清爽，吸收快。

✅ 真人味：挤出来是透明的凝露状，
          抹开有水珠爆开的感觉，
          嗖的一下就吸收了，
          完全不粘枕头！
```

### 4. 对话互动法

**公式**: 称呼 + 提问 + 回应预期

**示例**:
```
❌ AI 味：欢迎购买。

✅ 真人味：姐妹们！
          你们是不是也经常熬夜？
          这个真的要用起来！
          评论区告诉我效果！
```

### 5. 不完美真实法

**公式**: 优点 + 小缺点 + 总体推荐

**示例**:
```
❌ AI 味：完美无缺，强烈推荐。

✅ 真人味：说实话，包装一般般，
          但效果真的没话说！
          冲着效果可以忽略包装～
```

## 🎯 质量评估标准

### 真实性评分 (1-10 分)
- 9-10 分：完全像真人
- 7-8 分：大部分像真人
- 5-6 分：有明显 AI 痕迹
- 1-4 分：明显 AI 生成

### 亮点评分 (1-10 分)
- 9-10 分：多个记忆点
- 7-8 分：有 1-2 个亮点
- 5-6 分：平淡无奇
- 1-4 分：枯燥乏味

### 平台适配评分 (1-10 分)
- 9-10 分：完美适配
- 7-8 分：基本适配
- 5-6 分：需要调整
- 1-4 分：完全不适合

**发布标准**: 三项评分均≥7 分
