## 🤖 详细 Agent 设计

### 1. TrendHunter Agent (热点发现)

**职责**: 发现全网热点话题  
**输入**: 无 (定时触发)  
**输出**: 热点话题列表  

**调用 Skills/Tools**:
- `web_fetch` - 爬取热点页面
- `agent-browser` - 浏览器自动化
- `search_skills` - 搜索相关技能

**数据源**:
- 微博热搜
- 知乎热榜
- Twitter Trends
- Reddit Popular
- Google Trends

**输出示例**:
```json
{
  "trending_topics": [
    {
      "topic": "AI 发展趋势 2026",
      "heat_score": 95,
      "platforms": ["微博", "知乎", "Twitter"],
      "related_keywords": ["AGI", "大模型", "自动驾驶"]
    }
  ]
}
```

---

### 2. TopicRanker Agent (选题排序)

**职责**: 评估和排序选题  
**输入**: 热点话题列表  
**输出**: 优先级排序的选题建议  

**评估维度**:
- 热度分数
- 与账号定位匹配度
- 竞争程度
- 创作难度
- 预期流量

**输出示例**:
```json
{
  "recommended_topics": [
    {
      "topic": "AI 发展趋势 2026",
      "priority": "high",
      "reason": "热度高 + 匹配账号定位",
      "suggested_formats": ["文章", "视频", "图文"]
    }
  ]
}
```

---

### 3. ArticleWriter Agent (文章创作)

**职责**: 生成博客/公众号文章  
**输入**: 选题 + 大纲  
**输出**: 完整文章 (Markdown/Word)  

**调用 Skills**:
- `copywriting` - 文案写作
- `impeccable/polish` - 文章润色
- `docx` - Word 文档生成

**输出格式**:
- Markdown
- Word (.docx)
- PDF (可选)

---

### 4. ScriptWriter Agent (脚本创作)

**职责**: 生成视频脚本  
**输入**: 选题  
**输出**: 分镜脚本  

**脚本结构**:
```
场景 1 (0-5s): 开场钩子
场景 2 (5-15s): 问题提出
场景 3 (15-45s): 核心内容
场景 4 (45-60s): 总结 + CTA
```

**调用 Skills**:
- `remotion-best-practices` - 视频最佳实践
- `brainstorming` - 创意发散

---

### 5. Copywriter Agent (文案创作)

**职责**: 生成多平台文案  
**输入**: 选题  
**输出**: 各平台适配文案  

**平台适配**:

| 微博 | 140 字 | 简洁 | 多 |
| 小红书 | 500 字 | 亲切 | 很多 |
| Twitter | 280 字符 | 直接 | 中 |
| LinkedIn | 1000 字 | 专业 | 少 |

**调用 Skills**:
- `copywriting` - 文案写作
- `marketingskills` - 营销技能

---

### 6. ImageGenerator Agent (图像生成)

**职责**: 生成配图和封面  
**输入**: 文章/视频主题

### 7. VideoProducer Agent (视频制作)

**职责**: 制作完整视频  
**输入**: 脚本 + 图像 + 配音  
**输出**: 视频文件 (.mp4)  

**调用 Tools**:

**制作流程**:
```
脚本 → 分镜 → 素材收集 → 剪辑 → 特效 → 字幕 → 导出
```

## 📤 交付形式

### 1. 命令行交互

```bash
# 创建内容
claw content create --topic "AI 发展趋势" --formats article,video,copy

# 扫描热点
claw content trends --platforms weibo,zhihu,twitter

# 发布内容
claw content publish --file article.md --platforms wechat,weibo

# 查看分析
claw content analytics --days 7
```

### 2. 工作流自动化

```bash
# 运行完整工作流
python3 workflows/content_creator.py \
  --topic "AI 发展趋势" \
  --output-dir ./output \
  --platforms wechat,weibo,xiaohongshu \
  --auto-publish
```

### 3. 定时任务

```bash
# 每小时扫描热点
0 * * * * claw content trends --auto-save

# 每天发布内容
0 9 * * * claw content publish --scheduled
```
