**代表**: 科技袁老师、AI 研究所
**特点**:

**代表**: AI 视觉实验室、未来科技

- AI 生成图像/视频

- 科技感转场
- 赛博朋克风格
- 字幕大且醒目

#### 类型 C: PPT 动画 + 配音
**代表**: 科技最前线、数字生活
**特点**:
- 精美 PPT 动画
- 图标 + 文字动画
- 配色统一（蓝/紫科技感）
- 专业但不枯燥

## 🛠️ 技术实现

### 1. 素材准备

**背景视频**:
- 科技粒子背景
- 渐变流动背景
- 代码滚动背景

**图标库**:
- 工具图标
- 效率图标
- 时间图标

**BGM**:
- 轻快节奏
- 无版权
- 音量适中

### 2. 文字层生成

使用 FFmpeg drawtext:
```bash
ffmpeg -i background.mp4 \
  -vf "drawtext=text='AI 工具推荐':fontsize=50:fontcolor=white:x=(w-text_w)/2:y=(h-text_h)/2" \
  -c:v libx264 output.mp4
```

### 3. AI 配音

**推荐服务**:
-  Azure TTS（自然）
-  剪映配音（免费）
-  本地 TTS（离线）

### 4. 合成流程

```bash
# 1. 生成背景
ffmpeg -f lavfi -i color=c=#1a1a2e:s=1080x1920:d=60 background.mp4

# 2. 添加文字
ffmpeg -i background.mp4 -vf "drawtext=..." with_text.mp4

# 3. 添加配音
ffmpeg -i with_text.mp4 -i voiceover.mp3 -c:v copy -c:a aac with_voice.mp4

# 4. 添加 BGM
ffmpeg -i with_voice.mp4 -i bgm.mp3 -filter_complex "[0:a][1:a]amix=inputs=2" final.mp4
```
