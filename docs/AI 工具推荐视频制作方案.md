## 🛠️ 制作方法

### 方法 1: FFmpeg 素材合成（推荐）

**优点**: 快速、免费、可控
**缺点**: 需要 FFmpeg

**命令示例**:
```bash
# 1. 创建背景
ffmpeg -f lavfi -i color=c=#1a1a2e:s=1080x1920:d=60 bg.mp4

# 2. 添加文字
ffmpeg -i bg.mp4 -vf "drawtext=text='⚠️别再手动工作了！':fontsize=50:fontcolor=white:x=(w-text_w)/2:y=(h-text_h)/2" out.mp4

# 3. 添加配音
ffmpeg -i out.mp4 -i voiceover.mp3 -c:v copy -c:a aac final.mp4

# 4. 添加 BGM
ffmpeg -i final.mp4 -i bgm.mp3 -filter_complex "[0:a][1:a]amix=inputs=2" video.mp4
```

### 方法 2: PPT 动画转视频

**优点**: 信息清晰、易制作
**缺点**: 制作时间较长

**流程**:
1. 制作 PPT（16:9 或 9:16）
2. 添加动画效果
3. 导出为 MP4
4. 添加配音和 BGM

### 方法 3: HTML 动画转视频

**优点**: 灵活、可编程
**缺点**: 技术门槛

**工具**:
